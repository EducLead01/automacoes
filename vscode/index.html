Claude Code — Orquestração de Agentes

/effort

Ajusta a profundidade de raciocínio do modelo durante a própria conversa. Níveis: low, medium, high, xhigh, max, auto (auto volta ao padrão).

/effort ultracode

Nível especial: raciocínio xhigh + orquestração automática de fluxo entre agentes, para tarefas complexas de múltiplas etapas.

/code-review ultra

Revisão de código multi-agente na nuvem do branch atual — várias IAs revisam em paralelo e o resultado é consolidado. É paga e acionada manualmente.

/code-review ultra <PR#>

Mesma revisão multi-agente, aplicada a um Pull Request específico do GitHub em vez do branch local.

Boas Práticas EDUC LEAD

Um settings.json por projeto

Configure .vscode/settings.json com formatOnSave em cada pasta de cliente para manter formatação consistente.

Live Server para páginas HTML

Use a extensão Live Server para ver alterações nas páginas de clientes em tempo real, sem refresh manual.

Multi-cursor em vez de repetição

Ao editar blocos parecidos entre páginas de clientes, use Ctrl+D ou Ctrl+Shift+L.

Snippets personalizados

Em "Configure User Snippets", salve blocos repetidos entre projetos (ex: cabeçalho padrão de página de cliente).

.gitignore por projeto

Evite versionar arquivos temporários e pastas geradas antes de publicar no GitHub Pages.

Paleta de comandos como reflexo

Ctrl+Shift+P é o atalho universal — memorizar só esse já resolve boa parte do resto.
